# Simple Project To Do List

Basic Template for To Do List Project

###### Tech/framework used

JavaScript
Node.js
Embedded JavaScript (EJS)

###### clone this repository
###### First thing need to do
* npm init
* npm i express body-parser

###### To start up the server

* npm install -g nodemon
* nodemon app.js
